# UTL for Zed

[UTL (Universal Template Language)](https://bloxcms.com) support for the [Zed](https://zeditor.dev) editor. UTL is the template language used in the BLOX CMS platform by TownNews.

## Features

- Syntax highlighting via a [tree-sitter grammar](https://github.com/JekRock/tree-sitter-utl)
- HTML syntax highlighting injected into template content
- Bracket and quote auto-closing

## Installation

Search for **UTL** in Zed's extension registry (`zed: extensions`) and install it.
